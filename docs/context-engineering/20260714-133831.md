# Repository Context System Optimization Report

## Executive Summary

Verdict: `partially-agent-ready`

- Biggest strength: strong repo-local evidence already exists in `README.md`, `ROADMAP.md`, `docs/references/project-learnings.md`, `docs/decisions/2026-07-11-1545-rotation-capacity-scorer.md`, tests, and package scripts.
- Biggest risk: `AGENTS.md` is stale and overloaded, so future agents can start from the wrong storage path, command surface, and roadmap state.
- Top 5 improvements:
  1. Trim `AGENTS.md` to routing-only facts.
  2. Add a docs front door plus architecture/local-dev/testing docs.
  3. Split rotation behavior into a dedicated reference.
  4. Add context-quality tooling and CI.
  5. Create repo skills for repeat workflows.
- Expected context/token benefit: medium; repeated prompt context should drop because agents can route instead of rereading long prose.
- Expected reliability benefit: high; fewer stale assumptions, fewer doc/code mismatches, and more deterministic validation.

## Optimized AGENTS.md Proposal

```md
# MultiCodex Agent Notes

- Edit only files in this repository.
- Keep `/multicodex` as the only operator command surface. Do not reintroduce `/multicodex-use`, `/multicodex-status`, or `/multicodex-footer`.
- Use public exports from `@earendil-works/pi-ai`, `@earendil-works/pi-coding-agent`, and `@earendil-works/pi-tui`; avoid repo-local deep imports.
- Keep `README.md` user-facing, `ROADMAP.md` future-facing, and deeper canonical docs in `docs/`.
- Use `paths.ts` for all `~/.pi/agent/*` paths.
- When command UX, rotation behavior, storage schema, or release notes change, update `README.md`, `ROADMAP.md`, tests, and release notes together.
- Validate with:
  - `mise install`
  - `bun install`
  - `bun run lint`
  - `bun run tsgo`
  - `bun run test`
  - `bun run check`
  - `npm pack --dry-run`
  - `bun run release:dry`
  - `mise run pre-push`
- Prefer routing docs over prose. Read `docs/architecture.md`, `docs/local-development.md`, `docs/testing-reference.md`, `docs/references/project-learnings.md`, `docs/decisions/2026-07-11-1545-rotation-capacity-scorer.md`, and `docs/runs/<run-id>/...` when relevant.
- Missing context: `docs/README.md`, `docs/architecture.md`, `docs/local-development.md`, `docs/testing-reference.md`, `docs/context-engineering/README.md`, repo skills, and CI workflows are not yet present.
```
